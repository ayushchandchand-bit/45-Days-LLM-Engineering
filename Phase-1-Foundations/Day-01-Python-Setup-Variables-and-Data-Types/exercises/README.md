## Exercise 1 — Magic Trick 🎩
A classic "think of a number" trick. Start with **any** number and apply these steps:

1. Add 6
2. Multiply by 2
3. Subtract 4
4. Divide by 2
5. Subtract the original number

The result is **always 4**, no matter what number you start with!

**Your task:** in `magic_trick.py`, set a starting number in a variable, perform the five steps
using operators and assignment, and `print()` the final result. Try changing the starting
number — the answer should stay `4`.

➡ Solution: [`magic_trick_solution.py`](magic_trick_solution.py)

## Exercise 3 — Split the bill 🍕
A restaurant bill is `2400` rupees, split between `5` friends, with a `10%` tip on top.
Compute and print **how much each person pays** (tip included).

*Hint:* tip = `bill * 0.10`; total = `bill + tip`; per person = `total / friends`.

➡ Solution: [`split_bill_solution.py`](split_bill_solution.py)
